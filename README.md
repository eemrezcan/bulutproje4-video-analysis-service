# Video Analysis Service

Akilli Sehir Yonetim Platformu icin kamera ve video analiz servisidir.

## Sorumluluk

- Kamera noktalarini yonetir.
- Sentetik veya kayitli videoyu canli kamera gibi sunar.
- OpenCV/sentetik analiz ile recognition event'leri uretir.
- DynamoDB Local / Amazon DynamoDB uzerinde analiz event'lerini saklar.

## Lokal Port

Docker Compose ile `8002` host portundan erisilir.
